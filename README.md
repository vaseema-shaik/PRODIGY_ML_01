# House Price Prediction using Linear Regression

## Objective

This project predicts house prices based on:

* Square footage
* Number of bedrooms
* Number of bathrooms

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Machine Learning Algorithm

* Linear Regression

## Features Used

* GrLivArea
* BedroomAbvGr
* FullBath

## Output

The model predicts house prices and evaluates performance using Mean Absolute Error (MAE).
